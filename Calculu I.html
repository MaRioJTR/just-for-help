    Differentiation rules

    1. Constant Rule:

    • If c is a constant, then dy/dx(c) = 0.

    2. Power Rule:

    • If f(x) = xn, where n is a constant, then f'(x) = nxn-1.

    3. Sum/Difference Rule:

    • If f(x) = g(x) + h(x), then (ƒ(x)) = g'(x) + h'(x).

    4. Product Rule:

    • If f(x) = g(x) • h(x), then (ƒ(x)) = g(x) • h'(x) + h(x) • g'(x).

    5. Chain Rule:

    • If y = f(u) and u = g(x), then the derivative of y with respect to x is given by dy/dx = f'(u) * u'(x).

    Trigonometric functions

    1. Sine Function:

    (sin(x))' = cos(x)

    2. Cosine Function:

    (cos(x))' = -sin(x)

    3. Tangent Function:

    (tan(x))' = sec2(x)

    4. Cosecant Function:

    (csc(x))' = -csc(x) cot(x)

    5. Secant Function:

    (sec(x))' = sec(x) tan(x)

    6. Cotangent Function:

    (cot(x))' = -csc2(x)

    Inverse Trigonometric functions

    1. Inverse Sine Function (arcsin):

    (sin-1(x))' = 1 / √(1 - x2)

    2. Inverse Cosine Function (arccos):

    (cos-1(x))' = -1 / √(1 - x2)

    3. Inverse Tangent Function (arctan):

    (tan-1(x))' = 1 / (1 + x2)

    4. Inverse Cosecant Function (arccsc):

    (csc-1(x))' = -1 / (|x| √(x2 - 1))

    5. Inverse Secant Function (arcsec):

    (sec-1(x))' = 1 / (|x| √(x2 - 1))

    6. Inverse Cotangent Function (arccot):

    (cot-1(x))' = -1 / (1 + x2)
